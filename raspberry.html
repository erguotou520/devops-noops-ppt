什么是树莓派

一个只有信用卡大小的卡片式电脑

  • 迷你
  • Arm芯片
  • GPIO
  • 高扩展性
  • 可玩性

我可以用树莓派做什么

  • 一个完整的Linux系统

  • 网络收音机/智能音响

    叮当开源智能音箱

  • 无人机

  • 魔镜

还有什么

  • 绘画机/打字机

  • 遥控小车

  • 怀旧游戏机

  • Everything you can image

树莓派上都有些什么

怎么玩

  1. 有个点子/看到别人分享的好点子
  2. 刷相应的系统
  3. 可能需要学习一些Python代码来实现你的点子
  4. 注意安全

import RPi.GPIO as GPIO

import time

GPIO.setmode(GPIO.BOARD)
GPIO.setup(11, GPIO.OUT)
GPIO.output(11, True)
time.sleep(3)
GPIO.output(11, False)
GPIO.cleanup()

一个蜂鸣器demo

import time
import RPi.GPIO as GPIO
GPIO.setmode(GPIO.BOARD)
GPIO.setup(16, GPIO.OUT)

# 脉宽调制（PWM）
p = GPIO.PWM(16, 50)  # 通道为 16 频率为 50Hz
p.start(0)
try:
    while 1:
        for dc in range(0, 101, 5):
            p.ChangeDutyCycle(dc)
            time.sleep(0.1)
        for dc in range(100, -1, -5):
            p.ChangeDutyCycle(dc)
            time.sleep(0.1)
except KeyboardInterrupt:
    pass
p.stop()
GPIO.cleanup()
